\subsection{Für das Spiel registrieren}
\begin{description}
\item [Beschreibung] Der Spieler wählt einen Benutzernamen und ein Passwort. Ist der Benutzername noch nicht vergeben müssen weitere Angaben gemacht werden, um die Registrierung abzuschließen (Emailadresse, gewähltes Volk). Bei Abschluss der Registrierung erhält der Spieler eine Email mit einer Aktivierungsanweisung.
\item [Beispiel]
\item [Bemerkungen]
\item [Implementationsdetails]Der erste Teil der Reistrierung enthält einen Dialog mit zwei Textfelden, in denen der Spieler einen Benutzernamen und ein Passwort wählt. Nach dem Versenden der Informationen wird in der Datenbank überprüft, ob der Benutzername bereits vergeben ist, falls ja, muss der Spieler nachbessern. Kann der Name gewählt werden, müssen weitere relevante Informationen vergeben werden, die für das Spiel wichtig sind. Auch hierzu dient wieder ein Formular mit verschiedenen Text- und Auswahlfeldern. Die Informationen werden versammelt verschickt und in die Datenbank eingetragen. Der Spieler ist somit registriert und erhält eine Email mit einem Aktivierungslink.
\item [Aufgaben]
  \begin{itemize}
  \item Registrierungsseiten mit Eingabefeldern bauen
  \item Datenbankabfrage, ob Benutzer bereits vergeben
  \item Passwortkontrolle, ob beide Passwörter gleich
  \item übermittelte Daten in der Datenbank ablegen
  \item Aktivierungslink bauen und dem Spieler per Email zukommen lassen
  \item Bestätigungsseite anzeigen
  \end{itemize}
\end{description}
